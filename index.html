Como usar

  1. Obtenha o token do seu bot conversando com o @BotFather no Telegram
  2. Para obter o Chat ID de um grupo, adicione o @raw_data_bot ao grupo e copie o "id"
  3. Para canais públicos, use o nome do canal (ex: @nomedocanal)
  4. Certifique-se de que seu bot é administrador do grupo/canal
  5. Selecione os arquivos e clique em "Enviar para o Telegram"
